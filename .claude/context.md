## Technologies for Todo Full-Stack Web Application (Hackathon Phase 2)

### Backend Technologies:
- Python FastAPI: Web framework for building the API
- SQLModel: SQL Toolkit and ORM that combines SQLAlchemy and Pydantic
- PyJWT: For handling JWT token creation and verification
- Neon Serverless PostgreSQL: Cloud database service used for persistence

### Frontend Technologies:
- Next.js 16+: Framework for building the web application
- Better Auth: Authentication library for handling user registration/login
- App Router: Next.js routing system for navigation

### Security & Authentication:
- JWT (JSON Web Tokens): For authentication between frontend and backend
- Bearer Authentication: Standard for sending JWT in Authorization header
- Shared Secret: For signing and verifying JWT tokens

### API Design:
- RESTful API: Follows REST principles for endpoints
- HTTP Methods: GET, POST, PUT, DELETE, PATCH for CRUD operations
- Status Codes: Standard HTTP status codes (200, 201, 401, 403, 404, 500)
