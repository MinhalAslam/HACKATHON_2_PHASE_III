## Technologies for Frontend Web Application & API Integration

### Frontend Framework:
- Next.js 16+: React-based framework with App Router for modern web application
- App Router: Next.js routing system for navigation and page organization
- React 19: Frontend library for building user interfaces and components

### Styling & UI:
- Tailwind CSS: Utility-first CSS framework for responsive styling
- Responsive Design: Implementation for common screen sizes (mobile, tablet, desktop)

### HTTP & API Management:
- Axios: HTTP client library for API requests and response handling
- Request Interceptors: Automatic JWT token attachment to all requests
- Response Interceptors: Global handling of 401 Unauthorized responses

### Authentication & State Management:
- Better Auth: Authentication library for handling user signup/login
- React Context: Global state management for authentication state
- React Hooks: Built-in React state management (useState, useReducer, useEffect)
- React Query: Server state management for task data and API responses

### Form Handling:
- React Hook Form: Library for form handling with validation capabilities
- Form Validation: Built-in validation for authentication and task forms

### Types & Interfaces:
- TypeScript: Static typing for frontend components and API contracts
- Type Safety: Strict typing for all API requests and responses

### Component Architecture:
- Higher-Order Components (HOC): Pattern for route protection
- Custom Hooks: Encapsulation of authentication and API logic
- Reusable Components: Loading, error, and UI state components
